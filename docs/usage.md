# 使用指南

本文将以一段简单的汇编程序为例，介绍如何使用本工具。这段汇编程序是将内存中的数据复制到另一个内存地址中。

```assembly
.data
a:
.word 8

.text
la.local $r1, a
ld.w $r2, $r1, 0
addi.w $r1, $r1, 4
st.w $r2, $r1, 0
```
